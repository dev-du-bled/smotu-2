# smotu

Une capsule Lakebed pour deviner le mot du jour avec score ranked et classement quotidien.

## Lancer

```sh
npm install
npm run dev
```

Ouvrir `http://localhost:3000`.

Pour tester plusieurs joueurs en local:

```txt
http://localhost:3000/?lakebed_guest=alice
http://localhost:3000/?lakebed_guest=bob
```
